## 認証 / Authentication

全てのエンドポイントは`JWT`による認証トークンが必須です。
[ユーザーぺージ](https://hss.aknet.tech/users)からの取得が可能ですので、忘れることなく指定してください。

```header
Authorization : Bearer <UserToken>
```

## データの返り値

全てのエンドポイントは以下のデータ形式でレスポンスが返されます。

```json
{
    status : <number>
    body : {
        message : <nullable string>,
        data : <nullable any | any[]>
    }
}
```

※ statusの値は成功時は`0`、失敗時は`-1`が返されるように設定されています。
